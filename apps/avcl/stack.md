# Stack do projeto

- Backend em Django 5 com autenticação nativa do framework.
- Banco de dados relacional acessado por `DATABASE_URL`, usando SQLite para o ambiente de portfolio.
- Frontend server-rendered com Django Templates.
- Estilização com Tailwind CSS e DaisyUI via CDN.
- Visualização de indicadores com Chart.js.
- Automação assíncrona com Celery e `django-celery-beat`.
