# Control

Estudo de caso de um backoffice Django para gestão financeira recorrente, contratos de serviço, invoices e custos de infraestrutura associados aos clientes.

O ambiente de portfolio usa um banco SQLite demo separado com dados fictícios e acesso administrativo para navegação nas telas operacionais.

Telas destacadas:

- dashboard financeiro e operacional
- listagem de invoices
- contratos e clientes no admin
- visão mobile do dashboard
