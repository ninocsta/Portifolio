## Estrutura padrão de cada pasta

- `README.md`: visão geral do caso.
- `stack.md`: stack e arquitetura.
- `features.md`: funcionalidades centrais.
- `setup.md`: instruções para demo local.
- `screenshots/`: telas selecionadas.
