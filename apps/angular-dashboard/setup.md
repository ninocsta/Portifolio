# Setup local

## Pré-requisitos

- Node.js 18+
- Angular CLI: `npm install -g @angular/cli`

## Instalação

```bash
git clone https://github.com/ninocsta/Portifolio.git
cd Portifolio/apps/angular-dashboard
npm install
ng serve
```

Acesse `http://localhost:4200`.

## Notas

- Sem variáveis de ambiente necessárias
- A API REST Countries é pública e não exige chave
- O projeto usa dados reais da API em tempo real
