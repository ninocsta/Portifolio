# angular-dashboard

Projeto demonstrativo de front-end Angular — criado para mostrar domínio do framework fora do contexto corporativo.

Contexto: meu trabalho diário com Angular é em sistemas internos de empresa (Minesoft), cujo código não pode ser publicado. Este projeto reproduz padrões reais que uso no trabalho: componentes reutilizáveis, serviços, consumo de API REST, roteamento e tipagem com TypeScript.

## O que demonstra

- Componentes Angular com inputs/outputs tipados
- Serviço HTTP consumindo API pública (REST Countries)
- Roteamento com lazy loading
- Pipes personalizados
- Reactive Forms com validação
- Organização em módulos feature
