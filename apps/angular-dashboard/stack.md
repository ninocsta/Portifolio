# Stack

- Angular 17+ com standalone components
- TypeScript estrito (`strict: true` no tsconfig)
- RxJS para gerenciamento de estado assíncrono
- Angular HttpClient com tipagem forte
- Angular Router com lazy loading por rota
- Reactive Forms (FormBuilder, Validators)
- SCSS com variáveis e mixins
- API pública: REST Countries (https://restcountries.com) — sem necessidade de chave
