# Stack

- Django com arquitetura monolítica e renderização server-side
- autenticação customizada com `AUTH_USER_MODEL`
- multi-tenancy por domínio/subdomínio com middleware próprio
- SQLite demo isolado no portfolio
- Bootstrap 5 e templates customizados para site e portal
- Chart.js no dashboard do portal
- integração de billing modelada para Asaas
- Playwright para captura automatizada de screenshots
