# Stack

- Django com arquitetura monolitica e renderizacao server-side
- autenticacao customizada com `AUTH_USER_MODEL`
- multi-tenancy por dominio/subdominio com middleware proprio
- SQLite demo isolado no portfolio
- Bootstrap 5 e templates customizados para site e portal
- Chart.js no dashboard do portal
- integracao de billing modelada para Asaas
- Playwright para captura automatizada de screenshots
